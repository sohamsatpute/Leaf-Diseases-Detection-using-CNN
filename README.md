# 🌿 Leaf Disease Detection using CNN

## 📌 Project Overview
This project uses a Convolutional Neural Network (CNN) to detect plant leaf diseases from images. The model classifies leaves into healthy or diseased categories, helping farmers and agriculture experts take early action.

- Provide an automated and accurate solution
